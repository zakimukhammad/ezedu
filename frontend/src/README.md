# EzEdu Frontend Source Directory

Source code directory containing Astro pages, Preact components, styles, and client utilities.

---

## 📂 Subdirectories

- **`components/`**: Interactive client-side components built with Preact (login forms, quiz widgets, navigation, progress bars).
- **`layouts/`**: Astro layout templates used across pages (`BaseLayout.astro`, `AuthLayout.astro`).
- **`lib/`**: Frontend API client functions (`api.ts`), constants, and state management.
- **`pages/`**: File-based routes generated by Astro (`index.astro`, `masuk.astro`, `daftar.astro`, `profil.astro`).
- **`styles/`**: Custom CSS design tokens, typography, colors, and global stylesheets.
